# Tic Tac Toe Game

A simple web-based Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Demo

![Tic Tac Toe Game Screenshot](./screenshot.png)

> The above screenshot shows the initial interface of the game. The game board consists of a 3x3 grid, and players can take turns to play as "O" or "X". The UI features a soft pink background with rounded, shadowed tiles for a clean and pleasant look.

## Features

- **Interactive 3x3 Grid**: Play classic tic tac toe in the browser.
- **Player Turns**: Alternates automatically between "O" and "X".
- **Win Detection**: Alerts the winner when a player completes a row, column, or diagonal.
- **Draw Detection**: Announces a draw if all boxes are filled with no winner.
- **Reset/New Game Buttons**: Instantly start a new game or reset the current one.
- **Responsive Design**: Looks good on desktops and tablets.

## How to Play

1. Open `index.html` in your browser.
2. Click on any box to place your mark ("O" or "X").
3. Players alternate turns automatically.
4. The game detects wins and draws and displays the result.
5. Click **Reset Game** to clear the board or **New Game** to start over after a round.

## Usage

### Local Setup

1. Clone the repository or download the source code.
2. Make sure the following files are present:
    - `index.html`
    - `style.css`
    - `app.js`
3. *(Optional)* Save the screenshot from above as `screenshot.png` for the README image.
4. Open `index.html` in your preferred browser.
